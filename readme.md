## 📖 Project Overview
As the barbershop industry grows, relying on manual data entry leads to inefficiencies, data loss, and difficulties in tracking business performance. This project aims to digitalize barbershop management. 

Developed entirely in C, this system utilizes a **Binary Search Tree (BST)** to efficiently manage customer data, service histories, and barber performance. It provides a complete digital solution to replace manual bookkeeping, allowing owners to track daily operations and analyze revenue in real-time.

## ✨ Key Features
- **🚀 Advanced Data Structure:** Implements a Binary Search Tree (BST) for highly efficient data searching, insertion, and deletion.
- **📝 Full CRUD Operations:** Create, Read, Update, and Delete customer appointment and service records.
- **💇‍♂️ Comprehensive Service Categories:** Supports various service types with dynamic pricing:
  - Haircuts (Basic, Super, etc.)
  - Hair Dye
  - Perms
- **📊 Business Analytics Dashboard:** Automatically calculates:
  - Total Revenue generated.
  - Number of specific services rendered.
  - The "Most Popular Barber" based on customer frequency.
- **💾 Data Persistence:** Seamlessly saves and loads all transaction data to and from a local `barber_data.txt` file, ensuring no data is lost between sessions.
- **🎨 Interactive CLI:** Features a user-friendly, color-coded Command Line Interface.

## 🚀 How to Run Locally
### Prerequisites
Make sure you have a C compiler (like DevC++ or GCC) installed on your system.

## Step-by-Step Guide
1. Clone the Repository
    ```bash
    git clone https://github.com/yosukeyung/Barber-Shop-Management-C
    cd Barber-Shop-Management-C
    ```

2. Compile the Code
Open your terminal and run the following command to compile the C program:
    ```bash
    gcc finpro.c -o barbershop
    ```

3. Run the System
Execute the compiled program:
    ```bash
    barbershop.exe
    ```

4. Usage
Follow the on-screen menu to add customers, view tables, analyze data, and exit safely to save your records.
